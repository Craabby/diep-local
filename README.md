# Diep-local-C++
## The C++ rewirte of Diep-local

# NOT COMPILING
### undefined refrences

### dependencies
- websocket++
- boost

### building
```
mkdir build && cd build
cmake ..
make
# to install it, run `sudo make install`
```
### running
```
# make sure you are in the build directory
./diep-local
# if you installed it, run `diep-local`
```
